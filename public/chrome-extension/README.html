Install the Chrome extension

The AI Bookmark Organizer extension keeps your Chrome bookmarks and reading list in sync with this dashboard and adds AI-driven organisation.

  1. Download the repository or pull the latest changes so the chrome-extension folder is available locally.
  2. Open chrome://extensions, enable Developer mode, and choose Load unpacked.
  3. Select the chrome-extension directory from this project.
  4. Click the extension icon and sign in with your Supabase account. Grant access when prompted.
  5. Use Sync now or enable auto-sync to keep everything updated.

The extension communicates with /api/import-chrome using your Supabase access token. Make sure your Netlify environment has the correct Supabase and Gemini credentials configured.

For packaging instructions see the Chrome extension README.
